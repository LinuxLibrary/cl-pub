# EBS Backup

- Point-in-time snapshots to S3
- Snapshots can be used to:
	- Resize
	- Copy
	- Share
- Deleting snapshots only removes the data not needed by another snapshot
- Additional backup options:
	- VPN / Direct Connect
	- Agent based backup

# S3 Backup
